# PDF 本地去水印文字提取工具

本项目是一款纯本地运行的 PDF 文字提取与去水印桌面工具。

> [!IMPORTANT]
> 本产品处于 0.3.0 阶段。开发灵感源自于无外网连接的物理隔离办公环境，涉密数据无法上传云端，故基于本地原生 OCR 与私有大模型端点自研了本工具。

当前版本提供三类 PDF 场景处理、PDF 主画布、结果检查器、统一页码联动、本地 Vision OCR、本地 AI 端点净化与可见错误提示。

## 支持的 PDF 场景

工具按照真实办公中常见的三类 PDF 设计处理管线：

1. **电子文本 + 电子水印**：正文和水印都可以选中。工具读取 PDF 文本层，删除用户确认的高频电子水印词，不重新 OCR，准确率最高。
2. **扫描正文 + 电子水印**：正文是扫描图片，水印是后添加的可选中文字。工具默认保留原始扫描图像执行本地 Vision OCR，再过滤水印残留词；只有当电子水印本身严重干扰 OCR 时，才建议手动开启遮罩。
3. **全扫描件**：正文和水印都已经印在图像里。工具对整页图像执行 OCR，再按水印词做文本级过滤；若水印与正文像素严重重叠，建议继续使用本地 AI 净化辅助修复。

## 核心功能

1. 隐私保护：完全离线运行，使用 macOS 原生 Vision 文本识别，不上传任何数据。
2. 水印过滤：支持电子水印文本层清理、电子水印区域遮罩，以及 OCR 后残留词过滤。
3. 双通道提取：文字版 PDF 直接读取，扫描件 PDF 执行本地 Vision OCR。
4. 本地 AI 净化：支持连接 Ollama、LM Studio 等本地兼容端点进行排版纠错，支持选择是否开启留痕括号。
5. 联动界面：采用侧边栏 + PDF 主画布 + 结果检查器布局，支持 PDF 原件预览、原文、AI 净化按物理页码联动，支持本地另存为导出。

## 环境要求

- macOS 11.0 或更高版本
- 本地运行的 Ollama 或 LM Studio（如需开启 AI 净化功能）
- Xcode Command Line Tools（仅作为从源码编译的依赖，普通用户直接运行 App 无需安装）

## 启动与使用方式

直接双击运行项目目录下的 PDF文字提取.app 即可。

### 手动构建说明（仅供修改源码后使用）

如修改了 Sources/ 源码，可在终端运行以下命令重新打包：

```bash
chmod +x build.sh
./build.sh
```
