# PDF 本地去水印文字提取工具

本项目是一款纯本地运行的 PDF 文字提取与去水印桌面工具。

> [!IMPORTANT]
> 本产品处于 0.1.0 实验阶段。开发灵感源自于无外网连接的物理隔离办公环境，涉密数据无法上传云端，故基于本地原生 OCR 与私有大模型端点自研了本工具。

在 0.1.0 初始版本中，集成了本地 AI 净化助手，支持连接本地大模型服务进行排版、纠错与 Markdown 转换。

## 核心功能

1. 隐私保护：完全离线运行，使用 macOS 原生 Vision 文本识别，不上传任何数据。
2. 水印过滤：支持无损文本净化（精准替换）与细粒度字符擦除（像素遮盖）。
3. 双通道提取：文字版 PDF 直接读取，扫描件 PDF 自动擦除水印后执行 Vision OCR。
4. 本地 AI 净化：支持连接 Ollama、LM Studio 等本地兼容端点进行排版纠错，支持选择是否开启留痕括号。
5. 联动界面：采用四栏布局，支持 PDF 原件预览、原始识字、AI 净化三屏按物理页码双向翻页联动，支持本地另存为导出。

## 环境要求

- macOS 11.0 或更高版本
- 本地运行的 Ollama 或 LM Studio（如需开启 AI 净化功能）
- Xcode Command Line Tools（仅作为从源码编译的依赖，普通用户直接运行 App 无需安装）

## 启动与使用方式

直接双击运行项目目录下的 PDF文字提取.app 即可。

### 手动构建说明（仅供修改源码后使用）

如修改了 Sources/ 源码，可在终端运行以下命令重新打包：

```bash
chmod +x build.sh
./build.sh
```
