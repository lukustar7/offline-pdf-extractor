# PDF 本地去水印文字提取工具

这是一个专为 macOS 设计的 100% 离线、本地运行的 PDF 文字提取与去水印桌面工具。它完美解决了包含“活字水印”（可选中文字）的扫描版或文字版 PDF 的文字提取与水印净化问题，保护您的隐私不上传任何云端。

在 v1.1.0 版本中，我们集成了**本地 AI 净化助手**，支持一键连接您本地运行的模型服务，对提取出的原始文本进行自动排版与纠错。

## 🌟 核心功能

1. **绝对隐私保护**：完全在本地离线运行，使用 macOS 原生 Vision 文本识别引擎。连接本地 AI 时完全通过 `localhost` 环回接口通信，**不上传任何数据到互联网**，安全保密。
2. **智能水印识别与过滤**：
   - **自动扫描**：自动统计 PDF 前几页出现的高频词，推荐作为疑似水印词。
   - **双重保障去水印**：
     - *无损文本净化（默认）*：直接对提取文本或 OCR 文本进行水印词精准替换删除，绝对不影响或盖住正文扫描图像。
     - *细粒度字符擦除（可选）*：获取水印中每个字符的精确 bounds，在将页面转换为图片送去 OCR 时，用白色微斑遮住水印笔画，防止干扰 OCR。
3. **智能双通道文字提取**：
   - *文字版 PDF 页面*：直接提取正文并自动剔除水印活字，秒级完成。
   - *扫描件/图片 PDF 页面*：自动擦除水印后，调起高精度本地 Vision OCR 识别文字。
4. **✨ 本地 AI 净化排版与纠错 (新)**：
   - 支持 OpenAI 兼容的本地 AI 端点（如 **Ollama** 默认 `http://localhost:11434/v1` 或 **LM Studio** 默认 `http://localhost:1234/v1`）。
   - 一键获取本地可用模型列表并快速切换。
   - 内置精心设计的纠错提示词，强引导 AI 修复 OCR 别字、优化段落换行。
   - **直观错字对比**：AI 纠正的任何字词，都会在旁边紧随其后附上大括号，格式为 `【识别是：[错字]，修改为：[正字]】`，方便可读。
5. **现代化 UI 体验**：使用 SwiftUI 编写，支持深色模式、支持拖拽 PDF 文件导入、提供双栏对比 Tab 切换（原始文本 vs AI 净化文本）、提供运行日志和一键复制文本。

## 🛠️ 环境要求

- macOS 11.0 或更高版本
- Xcode Command Line Tools（需要 `swiftc` 编译器）
- 本地运行的 Ollama 或 LM Studio（如需开启 AI 净化功能）

## 🚀 启动与使用方式

您无需打开终端或懂任何编程，直接双击运行本项目目录下的 `PDF文字提取.app` 即可使用。

### 手动构建说明（若需重新编译）

如果您对 `Sources/` 目录下的源码进行了修改，可以在终端中运行以下命令重新编译打包：

```bash
chmod +x build.sh
./build.sh
```

构建成功后，将在当前目录下重新生成 `PDF文字提取.app`。
