# PDF 本地去水印文字提取工具

这是一个专为 macOS 设计的 100% 离线、本地运行的 PDF 文字提取与去水印桌面工具。它完美解决了包含“活字水印”（可选中文字）的扫描版或文字版 PDF 的文字提取与水印净化问题，保护您的隐私不上传任何云端。

## 🌟 核心功能

1. **绝对隐私保护**：完全在本地离线运行，使用 macOS 原生 Vision 文本识别引擎（Apple 神经引擎硬件加速），无任何网络请求。
2. **智能水印识别与过滤**：
   - **自动扫描**：自动统计 PDF 前几页出现的高频词，推荐作为疑似水印词。
   - **双重保障去水印**：
     - *无损文本净化（默认）*：直接对提取文本或 OCR 文本进行水印词精准替换删除，绝对不影响或盖住正文扫描图像。
     - *细粒度字符擦除（可选）*：获取水印中每个字符的精确 bounds，在将页面转换为图片送去 OCR 时，用白色微斑遮住水印笔画，防止干扰 OCR。
3. **智能双通道文字提取**：
   - *文字版 PDF 页面*：直接提取正文并自动剔除水印活字，秒级完成。
   - *扫描件/图片 PDF 页面*：自动擦除水印后，调起高精度本地 Vision OCR 识别文字。
4. **现代化 UI 体验**：使用 SwiftUI 编写，支持深色模式、支持拖拽 PDF 文件导入、提供流光进度条和提取文本预览，可一键复制或在 Finder 中查看导出的 TXT。

## 🛠️ 环境要求

- macOS 11.0 或更高版本
- Xcode Command Line Tools（需要 `swiftc` 编译器）

## 🚀 启动与使用方式

您无需打开终端或懂任何编程，直接双击运行本项目目录下的 `PDF文字提取.app` 即可使用。

### 手动构建说明（若需重新编译）

如果您对源码 `main.swift` 进行了修改，可以在终端中运行以下命令重新编译打包：

```bash
chmod +x build.sh
./build.sh
```

构建成功后，将在当前目录下重新生成 `PDF文字提取.app`。
